# Les cavernes de Mougevioù

## Général
Histoire Le donjon a été créé par des nains pour servir de repaire. Ses créateurs ont été détruits par une catastrophe naturelle, et le donjon a été conquis et modifié de nombreuses fois depuis lors.  
Murs Pierre taillée (DC 20 pour grimper)  
Sol Pierre naturelle (terrain difficile, désavantage sur les acrobaties)  
Température Moyenne  
Éclairage Sombre (les créatures individuelles peuvent porter des lampes)
