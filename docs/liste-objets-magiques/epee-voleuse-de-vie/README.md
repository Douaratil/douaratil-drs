Quand vous attaquez une créature avec cette arme magique et obtenez un cou critique sur le jet d'attaque, la cible subit 3d6 dégâts nécrotiques supplémentaires, à condition qu'elle ne soit ni une créature artificielle, ni un mort-vivant. Vous gagnez un nombre de points d'action temporaires égal aux dégâts supplémentaires infligés.  

**Fabrication**  
*Coût estimé*: Prix de l'épée x 50 po    
*Temps estimé*: Trois lunes  
*Points de pouvoir estimés*: 5000
