|**2**|Acide|
|**3**|Froid|
|**4-5**|Feu|
|**6**|Force|
|**7**|Foudre|
|**8**|Nécrotique|
|**9**|Poison|
|**10**|Psychique|
|**11**|Radiant|
|**12**|Tonnerre|
