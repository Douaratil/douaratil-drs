**Coût** : 4 points de Foi ou de Magie  
Le sorcier ouvre des plaies sur les deux poignets de sa cible. Celle-ci perd désormais 1d6 point d'action par round jusqu'à ce que des soins soient prodigués. Pour stopper le saignement, il faut faire un test de [Médecine](/utiliser-les-caracteristiques/#medecine) DD16. Une personne sans soins se vide de son sang et meurt doucement.  

**À plus haut niveau**.  
Si vous lancez ce sort en utilisant des niveaux de surcharge, les dégâts augmentent de 1d6 par niveau au-delà du niveau Novice.
